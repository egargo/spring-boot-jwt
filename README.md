## Dependencies

- [SDKMAN!](https://sdkman.io/)
    - Java
    - Maven

## Setup

```bash
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java
sdk install maven
```

## Run

```bash
mvn spring-boot:run
```
